必要環境

開発に必要な環境は以下の通りです。

  • Windows 2000/XP/Vista/7 (x86/x64)
  • FeliCaポート/パソリ(PaSoRi)
  • FeliCa ポートソフトウェア
  • 開発環境（コンパイラ)

 

FeliCaポート/パソリは RC-S330, RC-S320シリーズ、RC-S310/J1C, RC-S310/ED4C が使えるはずです。推奨は RC-S330です。

開発言語としては、C/C++ および C# が使えます。開発環境としては Visual Studio 2008 を推奨。個人的には Standard Edition をお勧めしますが、Express Edition でも動作すると思います(Platform SDK などのインストールは別途必要)。

x64 環境での注意事項はこちら。
