必要環境

開発に必要な環境は以下の通りです。

  • Windows 2000/XP/Vista
  • FeliCaポート/パソリ(PaSoRi)
  • FeliCa ポートソフトウェア
  • 開発環境（コンパイラ)

 

FeliCaポート/パソリは RC-S320シリーズ、RC-S310/J1C, RC-S310/ED4C が使えるはずです。推奨は RC-S320です。

開発言語としては、C/C++ および C# が使えます。開発環境としては Visual Studio 2005 を推奨。個人的には Standard Edition をお勧めしますが、Express Edition でも動作すると思います(Framework SDK などのインストールは別途必要)。
